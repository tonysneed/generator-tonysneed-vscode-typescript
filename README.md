## Tony Sneed's Yeoman Generator for TypeScript Projects Using Visual Studio Code

### Basic Usage

1. Install Yeoman

    ```shell
    npm install -g yo
    ```

2. Install the generator

    ```shell
    npm install -g generator-tonysneed-vscode-typescript
    ```

3. Use the generator

    ```shell
    yo tonysneed-vscode-typescript
    ```
